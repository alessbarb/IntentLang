# Global Flags

- [`--json`](#json) — Salida JSON estable para máquinas.
- [`--max-errors`](#max-errors) — Máximo de errores a imprimir en salida humana.
- [`--no-color`](#no-color) — Desactiva colores ANSI en la salida.
- [`--seed-clock`](#seed-clock) — Fija el reloj para builds/tests deterministas.
- [`--seed-rng`](#seed-rng) — Semilla RNG para test/build deterministas.
- [`--strict`](#strict) — Trata warnings como errores (falla con diagnósticos).
- [`--watch`](#watch) — Re‑ejecuta al cambiar archivos.

## --json {#json}

Salida JSON estable para máquinas.

| Name     | Type    | Default | Since | Deprecated |
| -------- | ------- | ------- | ----- | ---------- |
| `--json` | boolean | `false` | 0.2   | No         |

```bash
intent check packages/examples --json
```

> Exit codes: 0 ok, 1 lógicos/diags, 2 uso/flags/config (ILC040x).

## --max-errors {#max-errors}

Máximo de errores a imprimir en salida humana.

| Name           | Type   | Default | Since | Deprecated |
| -------------- | ------ | ------- | ----- | ---------- |
| `--max-errors` | number | —       | 0.2   | No         |

```bash
intent check src/**/*.il --max-errors 5
```

> No afecta salida JSON.

## --no-color {#no-color}

Desactiva colores ANSI en la salida.

| Name         | Type    | Default | Since | Deprecated |
| ------------ | ------- | ------- | ----- | ---------- |
| `--no-color` | boolean | `false` | 0.2   | No         |

## --seed-clock {#seed-clock}

Fija el reloj para builds/tests deterministas.

| Name           | Type   | Default                      | Since | Deprecated |
| -------------- | ------ | ---------------------------- | ----- | ---------- |
| `--seed-clock` | string | `fixed:1970-01-01T00:00:00Z` | 0.2   | No         |

```bash
intent build packages/examples --seed-clock fixed:2024-01-01T00:00:00Z
```

## --seed-rng {#seed-rng}

Semilla RNG para test/build deterministas.

| Name         | Type   | Default | Since | Deprecated |
| ------------ | ------ | ------- | ----- | ---------- |
| `--seed-rng` | string | `0`     | 0.2   | No         |

```bash
intent test --seed-rng 1234
```

## --strict {#strict}

Trata warnings como errores (falla con diagnósticos).

| Name       | Type    | Default | Since | Deprecated |
| ---------- | ------- | ------- | ----- | ---------- |
| `--strict` | boolean | `false` | 0.2   | No         |

```bash

```

## --watch {#watch}

Re‑ejecuta al cambiar archivos.

| Name      | Type    | Default | Since | Deprecated |
| --------- | ------- | ------- | ----- | ---------- |
| `--watch` | boolean | `false` | 0.2   | No         |

```bash

```

# Build Options

Opciones específicas para `intent build`.

- [`--out`](#out) — Directorio de salida para archivos emitidos.
- [`--sourcemap`](#sourcemap) — Emite sourcemaps (sólo cuando --target js).
- [`--target`](#target) — Formato de salida.

## --out {#out}

Directorio de salida para archivos emitidos.

| Name    | Type   | Default | Since | Deprecated |
| ------- | ------ | ------- | ----- | ---------- |
| `--out` | string | `dist`  | 0.2   | No         |

**Aliases:** `--outDir` · **Related:** [`--target`](#target), [`--sourcemap`](#sourcemap)

```bash
intent build src --out dist
```

> Alias soportado: --outDir.

## --sourcemap {#sourcemap}

Emite sourcemaps (sólo cuando --target js).

| Name          | Type    | Default | Since | Deprecated |
| ------------- | ------- | ------- | ----- | ---------- |
| `--sourcemap` | boolean | `false` | 0.2   | No         |

**Related:** [`--target`](#target)

```bash
intent build src -t js --sourcemap
```

## --target {#target}

Formato de salida.

| Name       | Type    | Default | Since | Deprecated |
| ---------- | ------- | ------- | ----- | ---------- | --- |
| `--target` | enum(ts | js)     | `ts`  | 0.2        | No  |

```bash
intent build src -t ts
```

```bash
intent build src -t js --sourcemap
```

# Test Options

Opciones específicas para `intent test`.

- [`--bail`](#bail) — Detiene la ejecución en el primer test fallido.
- [`--only`](#only) — Ejecuta sólo tests cuyo nombre hace match con el patrón.
- [`--reporter`](#reporter) — Formato de salida del runner.

## --bail {#bail}

Detiene la ejecución en el primer test fallido.

| Name     | Type    | Default | Since | Deprecated |
| -------- | ------- | ------- | ----- | ---------- |
| `--bail` | boolean | `false` | 0.2   | No         |

## --only {#only}

Ejecuta sólo tests cuyo nombre hace match con el patrón.

| Name     | Type   | Default | Since | Deprecated |
| -------- | ------ | ------- | ----- | ---------- |
| `--only` | string | —       | 0.2   | No         |

```bash
intent test src --only greet
```

## --reporter {#reporter}

Formato de salida del runner.

| Name         | Type       | Default | Since   | Deprecated |
| ------------ | ---------- | ------- | ------- | ---------- | --- |
| `--reporter` | enum(human | json)   | `human` | 0.2        | No  |

> Cuando se usa `--reporter json`, es equivalente a `--json` en cuanto a forma de salida.
